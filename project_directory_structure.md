# 工程配置目录的目录结构说明 #

## 目录结构 ##

### ./ ###

该目录下保存了工程的配置文件。

### ./release/ ###

该目录下保存了每个版本的更新信息和当前最新版本的更新信息。

更新信息文件命名规则为：**_[绝对版本号]_._[绝对子版本号]_.xml**

如果编译为加密版本，则文件名为：**_[绝对版本号]_._[绝对子版本号]_.xml.dat**

### ./release/res/ ###

该目录下保存了需要发布到更新服务器的更新文件。文件名取目标文件的相对路径的MD5值。

如果编译为加密版本，文件名取目标文件的相对路径加上一个密钥字符串后的MD5值。
